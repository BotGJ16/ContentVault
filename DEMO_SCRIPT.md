# ContentVault Demo Script

## Demo Overview
This script guides you through demonstrating ContentVault's key features for the Walrus Buildathon.

## Preparation
1. Ensure all services are running
2. Have test wallets ready with testnet ETH
3. Prepare sample content files
4. Set up recording software

## Demo Flow (5-7 minutes)

### 1. Introduction (30 seconds)
**Script**: 
"Hi everyone! I'm excited to present ContentVault, a decentralized content monetization platform built on the Walrus protocol. Today I'll show you how creators can monetize their content directly without intermediaries, using blockchain technology and decentralized storage."

**Screen**: Landing page with hero section

### 2. Wallet Connection (45 seconds)
**Script**:
"Let's start by connecting our wallet. ContentVault supports multiple wallets through Web3Modal. I'll connect my MetaMask wallet to access the platform features."

**Actions**:
- Click "Connect Wallet"
- Select MetaMask
- Approve connection
- Show wallet address and balance

**Screen**: Wallet connection flow

### 3. Creator Dashboard (60 seconds)
**Script**:
"This is the creator dashboard where you can manage your content, track earnings, and view analytics. As you can see, we have a clean, intuitive interface designed for both technical and non-technical users."

**Actions**:
- Navigate to dashboard
- Show earnings overview
- Display content statistics
- Show token balance

**Screen**: Creator dashboard with metrics

### 4. Content Upload (90 seconds)
**Script**:
"Now let's upload some content. I'll upload a sample video file. ContentVault automatically encrypts the file client-side before uploading to Walrus, ensuring complete privacy and security."

**Actions**:
- Click "Upload Content"
- Select video file
- Add metadata (title, description, price)
- Set access permissions
- Upload and show progress

**Screen**: Upload interface with encryption indicator

**Key Points**:
- Client-side encryption
- Walrus decentralized storage
- Automatic thumbnail generation
- Metadata storage on-chain

### 5. Content Discovery (60 seconds)
**Script**:
"ContentVault uses AI to recommend relevant content to users. Our recommendation engine analyzes user behavior, preferences, and interactions to provide personalized content feeds."

**Actions**:
- Navigate to explore page
- Show AI recommendations
- Filter by content type
- Search functionality

**Screen**: Content grid with AI recommendations

### 6. Payment & Access (75 seconds)
**Script**:
"Let's purchase access to some premium content. I'll buy this video for 0.01 ETH. The payment goes directly to the creator with only a 5% platform fee - much better than traditional platforms that take 30-45%."

**Actions**:
- Select premium content
- Click "Purchase Access"
- Confirm transaction in MetaMask
- Show transaction receipt
- Access decrypted content

**Screen**: Payment flow with MetaMask popup

**Key Points**:
- Direct creator payments
- Low platform fees (5%)
- Instant access after payment
- Automatic decryption

### 7. Tipping Feature (45 seconds)
**Script**:
"Users can also tip creators directly. This creates additional revenue streams and builds stronger creator-fan relationships. Let's send a small tip to show appreciation."

**Actions**:
- Click "Tip Creator"
- Enter tip amount
- Confirm transaction
- Show updated earnings

**Screen**: Tipping interface

### 8. Analytics & Earnings (60 seconds)
**Script**:
"Creators have access to detailed analytics showing their earnings, content performance, and audience insights. All transactions are transparent and verifiable on-chain."

**Actions**:
- Navigate to analytics page
- Show earnings breakdown
- Display content performance
- Show transaction history

**Screen**: Analytics dashboard with charts

### 9. Governance & Tokens (45 seconds)
**Script**:
"ContentVault has a governance system where users earn Creator Tokens (CTV) for platform participation. These tokens can be used for voting on platform decisions and accessing premium features."

**Actions**:
- Show token balance
- Display governance proposals
- Show voting interface
- Explain token utility

**Screen**: Governance dashboard

### 10. Technical Architecture (90 seconds)
**Script**:
"Let me quickly explain the technical architecture. ContentVault uses React.js for the frontend, Node.js for the backend, and Solidity smart contracts. The key innovation is our integration with Walrus for decentralized storage - content is encrypted client-side, stored on Walrus, and access is controlled by smart contracts."

**Key Points**:
- React.js + Next.js frontend
- Node.js + Express backend
- Solidity smart contracts
- Walrus decentralized storage
- AI recommendation engine
- End-to-end encryption

### 11. Why Walrus (60 seconds)
**Script**:
"We chose Walrus because it provides the perfect combination of decentralized storage, programmable access control, and cost-effective content delivery. Unlike traditional storage solutions, Walrus gives us censorship resistance, global CDN capabilities, and seamless integration with blockchain-based access controls."

**Key Benefits**:
- Decentralized and censorship-resistant
- Programmable access controls
- Global content delivery
- Cost-effective storage
- Native blockchain integration

### 12. Conclusion (30 seconds)
**Script**:
"ContentVault represents the future of content creation - where creators have full control over their content, receive fair compensation, and build direct relationships with their audience. Thank you for watching, and I'm happy to answer any questions!"

## Recording Tips

### Technical Setup
- Use 1080p resolution minimum
- Record at 30fps
- Clear audio narration
- Highlight mouse clicks
- Use smooth transitions

### Content Strategy
- Keep each segment focused
- Show real transactions
- Use actual content files
- Demonstrate error handling
- Show loading states

### Engagement
- Speak clearly and confidently
- Maintain good pace
- Show enthusiasm
- Make eye contact (if on camera)
- Use hand gestures appropriately
